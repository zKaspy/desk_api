## Доска объявлений

### Инструкция по запуску:

#### 1) Скачать архив и распаковать 
#### 2) Открыть проект в Docker контейнере 
```
docker-compose up -d --build
```
#### 3) Запустить миграции
```
docker-compose exec php-fpm bash
php artisan migrate --seed
```
#### 4) Открыть сайт 
```
localhost/
```
